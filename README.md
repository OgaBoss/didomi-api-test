## Didomi Backend Api Challenge

## Set Up

```bash
$ npm install

Copy .env.example into your .env file.

Fill out the variables as required.
```

## Running the app

```bash
# development
$ npm run start

```

## DB
```
Database used is mysql
```

## API Docs
```
When the app is running, check http://localhost:3000/api/ for API docs
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
